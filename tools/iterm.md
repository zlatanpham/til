# iTerm

## Adding useful shortcuts

1. Preferences \(âŒ˜ ,\) &gt; Profiles &gt; Keys
2. Click the plus.

**Move forward one word**

```text
option+right
send escape
f
```

**Move back one word**

```text
option+left
send escape
b
```

**Delete to beginning of word**

```text
option+delete
send hex code
0x1B 0x08
```

**Delete to end of word**

```text
fn+option+delete
send escape
d
```
